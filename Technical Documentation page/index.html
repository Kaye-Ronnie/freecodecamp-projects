Variables

Variables just like in other programming languages, its role is to remember a value from some part of program. Technically they store valuables. Variables are declared using assignment operator (=) i.e Age = 20;

Variables have a few rules to be followed

  • variable names can contain letters, numbers, and the underscore
  • Variable names cannot contain spaces. for instance first name but first_name/first-name
  • Variable names cannot start with a number.
  • Case sensitve for instance, age and Temp Age are different.
pi = int(22 / 7); //pi is variable with value of got from solution
Diameter = 14; // Diameter is variable storing a value 14
radius = Diameter/2;
Area = pi *(radius**2)
print('Area is ', Area)

If Statements

Quite often in programs we only want to do something provided something else is true. Python’s if statement is what we need

If statements work with conditional operators and some are listed below

if [condition] :
{statement/output}
else: [statement/output]

  • if p > 4 : if p is greater than 4
  • if p >= 4 : if p is greater than or equal 4
  • if p <= 4 : if p is less than or equal to 4
  • if p == 4 : if p is equal to 4 (for equal to ; use == not =)
  • if p != 4 : if p is not equal to 4
if p > 4:
print('p is greater than 4');
else:
Print('P is less than or equal to 4');
